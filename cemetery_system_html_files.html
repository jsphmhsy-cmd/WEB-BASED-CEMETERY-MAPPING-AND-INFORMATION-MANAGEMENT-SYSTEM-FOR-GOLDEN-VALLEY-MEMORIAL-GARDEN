Project Summary

This repository contains a simple static front-end prototype of the Web-based Cemetery Mapping and Information Management System (based on the PDF you uploaded). Use these pages as UI templates when building your real app and backend.

Modules Included (from the PDF)

  • Login Module (Admin / Staff)
  • User Account Management
  • Burial Information
  • Lot Information
  • Mapping / Interactive Map
  • Billing & Payments
  • Reports & Export
  • Interment Scheduling
  • Lot Transfer
  • Visitor Portal

How to deploy on GitHub Pages

  1. Create a new repository on GitHub.
  2. Clone it locally and copy the files in this project (index.html, admin.html, staff.html, visitor.html, mapping.html, styles.css, script.js).
  3. Commit and push.
  4. On GitHub, go to Settings → Pages and set the branch (main) and root folder (/) then save.
  5. Visit https://<your-username>.github.io/<repo-name>/.
